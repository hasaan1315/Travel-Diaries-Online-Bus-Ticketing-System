<h2>Description</h2>
Travel Diaries is a Java-based desktop application designed to streamline the bus ticket reservation process. It enables passengers to register/login, enter personal and travel details, choose specific seats, and securely complete payments. Developed with Java Swing and integrated with MS Access databases, the system offers a practical solution for digitalizing traditional bus ticketing workflows.

<h2>Languages and Technologies Used</h2>
- <b>Java (Swing, AWT)</b>  
- <b>MS Access (UCanAccess JDBC Driver)</b>  
- <b>JDBC</b>  
- <b>NetBeans / Eclipse / IntelliJ</b> (Java IDE)

<h2>Environments Used</h2>
- <b>Windows OS</b>  
- <b>Java Runtime Environment</b>  
- <b>UCanAccess JDBC Connector</b>
